# Brick Breaker

A Brick Breaker game built with Object-Oriented Programming in JavaScript using the HTML Canvas element.

## Learning Objectives

- Build a complete Brick Breaker game using Object-Oriented Programming
- Use the HTML Canvas element for 2D game rendering
- Apply inheritance, encapsulation, and method overriding in JavaScript

## Getting Started

Install dependencies and start the dev server:

```bash
pnpm install
pnpm dev
```
